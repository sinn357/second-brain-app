# Project Ideas Collection

## 🎯 포커스/시간 관리

### Async Communication Hub
- 즉각 답변 불필요한 질문/업데이트 모음
- 정해진 시간에만 확인 (알림 X)
- 팀원들과 집중 시간 존중하는 문화 만들기

## 📝 노트/정보 관리

### Quick Capture Inbox
- 전역 단축키로 즉시 메모 (앱 전환 없이)
- 나중에 정리할 임시 메모함 (inbox zero 철학)
- 음성 메모도 자동 텍스트 변환

### Reading Queue Manager
- 나중에 읽을 아티클 저장 (Pocket 같은데 더 스마트하게)
- 읽는데 걸리는 시간 예측
- 카테고리별 자동 분류 + AI 요약

### Link Rot Detector
- 저장한 링크들 정기적으로 체크 (404 감지)
- 중요한 페이지는 자동 아카이빙
- 중복 북마크 찾기

### Meeting Notes Assistant
- 회의 중 빠른 노트 템플릿 (참석자, 결정사항, 액션아이템)
- 액션아이템 자동 추출 → 태스크로 변환
- 다음 회의 때 이전 액션아이템 체크

### Research Paper Manager
- PDF 논문 자동 메타데이터 추출
- 하이라이트/주석 관리
- 인용 생성 (APA, MLA 등)

### Daily Reflection Prompt
- 매일 저녁 알림: "오늘 배운 것 3가지는?"
- 시간이 지나면 과거 회고 랜덤 표시
- 성장 패턴 추적

## 💪 습관/목표

### Micro Goals
- 큰 목표를 작은 단위로 쪼개기
- "책 쓰기" → 매일 200단어 쓰기 같은 식으로 자동 분해
- 진행률 시각화

### Habit Stacking Builder
- 기존 습관에 새 습관 붙이기 ("커피 마신 후 → 10분 독서")
- 루틴 체인 시각화
- 체인이 끊어진 지점 분석

### Accountability Partner Matcher
- 비슷한 목표 가진 사람들 연결
- 서로 진행상황 공유/체크
- 주간 체크인 자동 스케줄링

### Goal Decomposer
- "마라톤 완주" → 주별 러닝 거리 증가 계획
- SMART 목표 프레임워크 자동 적용
- 중간 마일스톤 자동 생성

### Win Journal
- 매일 작은 승리 3개 기록
- 힘들 때 과거 승리들 보여주기
- 월간 하이라이트 자동 생성

## 🔧 개발자 도구

## 🎮 게임 아이디어

### 1️⃣ 중독성 멀티플레이어 소셜 + 캐주얼/모바일

#### Auto Chess Party (오토배틀 파티)
- **컨셉**: 5분 완결 오토체스, 친구들과 실시간 8인 대전
- **중독 요소**: 시즌 패스 + 데일리 미션, 친구와 듀오 모드, "한 판만 더" 싶게 짧은 플레이타임
- **소셜**: 리플레이 공유, 친구 덱 따라하기, 길드 토너먼트

#### Rhythm Battle Royale
- **컨셉**: 리듬게임 + 배틀로얄 (30명 동시 참여)
- **중독 요소**: 노래 해금 시스템, 스킨/이펙트 수집, 리더보드 + 랭크 시스템
- **소셜**: 팀 모드, 클랜전, 곡 만들어서 공유

#### Pet Battle Arena (펫 배틀 아레나)
- **컨셉**: 포켓몬 스타일 펫 수집 + 실시간 PvP
- **중독 요소**: 가챠 펫 수집, 펫 육성/진화, 주간 이벤트 펫
- **소셜**: 펫 교환, 친구 배틀, 길드 레이드

#### Quick Draw Duel (퀵드로우 듀얼)
- **컨셉**: 1대1 빠른 반응 미니게임 대결
- **중독 요소**: 매치메이킹 (실력 비슷한 사람), 승률 추적, 미니게임 종류가 계속 추가됨
- **소셜**: 친구 초대 대결, 토너먼트 브래킷

#### Island Merge & Raid (아일랜드 합치기 & 습격)
- **컨셉**: Merge 게임 + 다른 플레이어 섬 습격
- **중독 요소**: 아이템 합치기 (2048 스타일), 섬 꾸미기, 방어 타워 설치
- **소셜**: 친구 섬 방문, 선물 주고받기, 협동 레이드

### 2️⃣ 2D 도트 인디/아트 게임 (스토리 중심)

#### The Last Lighthouse Keeper
- **스토리**: 세상 마지막 등대지기. 바다에서 오는 "무언가"를 막아야 함
- **게임플레이**: 낮(등대 수리, 자원 수집, NPC와 대화) / 밤(타워 디펜스처럼 방어)
- **감성**: 고독, 책임, 희망
- **아트**: 픽셀 아트 + 날씨/시간 변화로 분위기 연출

#### Letters from the Past
- **스토리**: 할머니 유품에서 발견한 편지. 과거와 현재를 오가며 가족사 발견
- **게임플레이**: 과거(1940년대 전쟁 시절 생존) / 현재(탐정처럼 단서 모으기) / 두 시간대가 서로 영향
- **감성**: 가족, 기억, 역사
- **아트**: 과거는 세피아톤, 현재는 컬러풀

#### The Gardener's Dream
- **스토리**: 꿈속 정원에 갇힌 정원사. 각 식물이 기억을 담고 있음
- **게임플레이**: 식물 키우기 + 퍼즐, 식물마다 미니 스토리, 정원 꾸미기로 잃어버린 기억 복원
- **감성**: 치유, 상실, 재생
- **아트**: 파스텔 픽셀, 몽환적 색감

#### Echoes of the Station
- **스토리**: 폐쇄된 우주 정거장. AI와 단둘이 남음. 무슨 일이 있었나?
- **게임플레이**: 메트로이드바니아 탐험, AI와 대화하며 진실 발견, 선택에 따라 다른 엔딩
- **감성**: 고립, 신뢰, 진실
- **아트**: 사이버펑크 픽셀, 네온 조명

#### A Painter's Journey
- **스토리**: 기억을 잃은 화가. 그림을 그리면 기억 일부가 돌아옴
- **게임플레이**: 실제로 픽셀 아트 그리기(간단한 툴), 그린 그림이 현실에 영향(다리 그리면 다리 생김), 수집한 색상으로 세계 복원
- **감성**: 창작, 정체성, 색채
- **아트**: 흑백에서 시작해 점점 컬러풀하게

#### The Forgotten Town
- **스토리**: 지도에 없는 마을. 주민들은 모두 과거에 갇혀 있음
- **게임플레이**: 각 주민의 과거 사건 해결, 타임루프 퍼즐, 모든 걸 해결하면 마을이 "해방"
- **감성**: 그리움, 집착, 해방
- **아트**: Earthbound 스타일 픽셀

### 3️⃣ 젤다 BotW 스타일 3D RPG

#### Nomad's Compass (유목민의 나침반)
- **컨셉**: 광활한 사막 탐험. 방향만 알려주는 나침반 하나
- **BotW 요소**: 완전 오픈월드(어디든 갈 수 있음), 물리 엔진(모래폭풍, 신기루, 오아시스), 자유도 높은 문제 해결(등반, 글라이딩, 낙타 타기)
- **차별점**: 생존 요소(물, 체온 관리), 모래 속 고대 유적 발굴, 동적 날씨(모래폭풍 속 전투)

#### Sky Archipelago (하늘 군도)
- **컨셉**: 떠다니는 섬들의 세계. 비행선으로 탐험
- **BotW 요소**: 섬마다 고유 생태계/문화, 자유로운 전투 스타일(근접, 원거리, 마법), 환경 이용(바람, 중력, 번개)
- **차별점**: 비행선 커스터마이징, 3D 공중전, 섬과 섬 사이 탐험의 긴장감

#### The Wild Frontier (야생의 개척지)
- **컨셉**: 미개척 대륙. 문명 없는 원시 자연
- **BotW 요소**: 사냥, 채집, 크래프팅, 동물 조련(말, 늑대, 곰 등), 역동적 생태계(먹이사슬, 계절 변화)
- **차별점**: 캠프 건설, NPC 없음(진짜 혼자), 환경이 스토리(유적 발견으로 과거 문명 이해)

#### Elemental Highlands (원소의 고지대)
- **컨셉**: 4개 원소 지역(불, 물, 바람, 땅). 각각 완전히 다른 룰
- **BotW 요소**: 원소 조합(불+풀=산불, 물+전기=감전), 다양한 도구/무기, 논리적 퍼즐
- **차별점**: 원소 지역마다 다른 플레이 스타일, 원소 변신 능력, 지역 경계에서 원소 충돌

### 4️⃣ 미래 완전 몰입형 가상세계 (아이디어만)

#### Second Life, but Real
- **컨셉**: 진짜로 "제2의 인생" 살기
- **미래 기술**: 뇌-컴퓨터 인터페이스(생각만으로 조작), 촉각/후각/미각 피드백, 시간 지각 조작(가상에선 1년이 현실 1일)
- **가능성**: 다른 인생 체험(다른 성별, 나이, 능력), 불가능한 경험(우주 여행, 과거 시대), 완전한 창작 자유(세계 자체를 만들기)

#### Collective Dream World
- **컨셉**: 모두가 공유하는 꿈 세계
- **미래 기술**: 수면 중 접속, 잠재의식 반영(당신의 감정이 날씨에 영향), 다른 사람 꿈 방문
- **가능성**: 악몽 치료, 집단 창작(여러 사람 꿈이 합쳐짐), 꿈 기록/재생

#### Memory Reliving
- **컨셉**: 자신의 기억 속으로 들어가기
- **미래 기술**: 기억 추출/재생, 기억 속 다른 선택 해보기, 타인 기억 체험(동의 하에)
- **가능성**: 트라우마 치료, 역사 교육(역사 인물 시점), 사라진 사람과 재회

#### The Infinite Library
- **컨셉**: 모든 지식/이야기가 있는 무한 도서관
- **미래 기술**: AI 생성 무한 콘텐츠, 책 속으로 들어가기, 지식 직접 업로드(매트릭스처럼)
- **가능성**: 순간 학습, 모든 이야기 체험, 지식 공유 네트워크

#### Parallel Worlds Traveler
- **컨셉**: 평행우주 여행
- **미래 기술**: 양자 시뮬레이션, "만약에" 시나리오 탐험, 다른 버전의 나 만나기
- **가능성**: 결정 후회 해소, 무한 가능성 탐험, 자기 이해(다른 선택의 나)

#### Hive Mind Game
- **컨셉**: 여러 사람이 하나의 의식처럼 플레이
- **미래 기술**: 생각 공유, 집단 지성 체험, 개성 유지하면서 연결
- **가능성**: 완벽한 팀워크, 새로운 형태의 소통, 인간 의식의 확장

## 📊 데이터 시각화

### Life Dashboard (인생 대시보드)
- 건강(수면, 운동, 식사), 재무(수입/지출), 시간(어디에 썼나) 통합
- 각 영역 간 상관관계 발견 (잠 적게 자면 생산성 떨어짐)
- 인터랙티브 타임라인

### Personal Data Archive (개인 데이터 아카이브)
- 모든 디지털 흔적 시각화 (SNS, 이메일, 브라우저 히스토리)
- "지난 10년간 당신" 스토리텔링
- 프라이버시 중심 (로컬 처리)

### Habit Heatmap (습관 히트맵)
- GitHub 잔디밭 스타일이지만 더 상세
- 여러 습관 레이어로 보기
- 패턴 자동 감지 (주말에 운동 안 함 등)

### Investment Portfolio 3D (투자 포트폴리오 3D)
- 자산들을 3D 공간에 배치 (위험도, 수익률, 비중)
- 리밸런싱 시뮬레이션
- 역사적 백테스팅 애니메이션

### Test Coverage Map (테스트 커버리지 맵)
- 코드를 지도처럼 (테스트된 곳=밝게, 안 된 곳=어둡게)
- 인터랙티브 드릴다운
- "위험 지역" 하이라이트

### Social Network Graph (소셜 네트워크 그래프)
- 친구/지인들 관계 시각화
- 중심성, 클러스터 분석
- "다리 역할" 하는 사람 찾기
- 시간에 따른 네트워크 변화

### Conversation Analyzer (대화 분석기)
- 메신저 대화 시각화
- 누가 먼저 말 거나, 응답 속도, 주제 변화
- 워드클라우드, 감정 분석
- 관계 건강도 점수

### Learning Tree (학습 트리)
- 배운 것들을 트리 구조로
- 선수 지식 화살표
- 진행도 색상 코딩
- 다음 배울 것 추천

### Book Connection Map (책 연결 맵)
- 읽은 책들 사이 연결 (비슷한 주제, 인용)
- 클러스터로 관심사 파악
- 다음 책 추천

### Sleep Cycle Waves (수면 주기 웨이브)
- 수면 단계를 파도처럼
- 꿈 기록 오버레이
- 수면 질 점수 트렌드

### Mood Calendar (기분 캘린더)
- 매일 기분을 색상으로
- 날씨, 일정, 활동과 상관관계
- 패턴 예측 (다음 주 힘들 수 있음)

### Music Listening Universe (음악 청취 우주)
- 들은 노래/아티스트를 별자리처럼
- 장르별 은하
- 시간 여행 (과거 취향 탐험)

### Color Palette Extractor (컬러 팔레트 추출기)
- 이미지/영상에서 색상 추출
- 시간에 따른 색상 변화
- 영화 전체의 "색상 시그니처"

### Writing Rhythm (글쓰기 리듬)
- 글쓰기 세션 시각화
- 타이핑 속도, 삭제/수정 패턴
- 집중 구간 vs 고민 구간
