# Project Ideas Collection

## 🎯 포커스/시간 관리

### Async Communication Hub
- 즉각 답변 불필요한 질문/업데이트 모음
- 정해진 시간에만 확인 (알림 X)
- 팀원들과 집중 시간 존중하는 문화 만들기

## 📝 노트/정보 관리

### Quick Capture Inbox
- 전역 단축키로 즉시 메모 (앱 전환 없이)
- 나중에 정리할 임시 메모함 (inbox zero 철학)
- 음성 메모도 자동 텍스트 변환

### Reading Queue Manager
- 나중에 읽을 아티클 저장 (Pocket 같은데 더 스마트하게)
- 읽는데 걸리는 시간 예측
- 카테고리별 자동 분류 + AI 요약

### Link Rot Detector
- 저장한 링크들 정기적으로 체크 (404 감지)
- 중요한 페이지는 자동 아카이빙
- 중복 북마크 찾기

### Meeting Notes Assistant
- 회의 중 빠른 노트 템플릿 (참석자, 결정사항, 액션아이템)
- 액션아이템 자동 추출 → 태스크로 변환
- 다음 회의 때 이전 액션아이템 체크

### Research Paper Manager
- PDF 논문 자동 메타데이터 추출
- 하이라이트/주석 관리
- 인용 생성 (APA, MLA 등)

### Daily Reflection Prompt
- 매일 저녁 알림: "오늘 배운 것 3가지는?"
- 시간이 지나면 과거 회고 랜덤 표시
- 성장 패턴 추적

## 💪 습관/목표

### Micro Goals
- 큰 목표를 작은 단위로 쪼개기
- "책 쓰기" → 매일 200단어 쓰기 같은 식으로 자동 분해
- 진행률 시각화

### Habit Stacking Builder
- 기존 습관에 새 습관 붙이기 ("커피 마신 후 → 10분 독서")
- 루틴 체인 시각화
- 체인이 끊어진 지점 분석

### Accountability Partner Matcher
- 비슷한 목표 가진 사람들 연결
- 서로 진행상황 공유/체크
- 주간 체크인 자동 스케줄링

### Goal Decomposer
- "마라톤 완주" → 주별 러닝 거리 증가 계획
- SMART 목표 프레임워크 자동 적용
- 중간 마일스톤 자동 생성

### Win Journal
- 매일 작은 승리 3개 기록
- 힘들 때 과거 승리들 보여주기
- 월간 하이라이트 자동 생성

## 🔧 개발자 도구

### Side Project Dashboard
- 여러 프로젝트 한눈에 보기
- 각 프로젝트별 마지막 커밋, 이슈 수, 배포 상태
- "2주 이상 방치된 프로젝트" 알림

### Code Review Checklist
- PR 올리기 전 자동 체크 (테스트 있나? 문서 업데이트했나?)
- 팀별 커스텀 체크리스트
- 놓친 항목 하이라이트
