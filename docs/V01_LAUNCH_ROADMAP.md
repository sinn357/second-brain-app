## 🎯 제품 정의

### 핵심 가치 제안

```
"AI가 당신의 생각을 연결하고, 새로운 아이디어를 창조합니다"

- 흩어진 노트를 AI가 자동으로 연결
- 잊혀진 아이디어를 적시에 꺼내줌
- 지식의 빈틈을 발견하고 채워줌
```

### 타겟 사용자

**Primary (핵심)**
| 페르소나 | 핵심 니즈 | 킬러 기능 |
|----------|----------|----------|
| 연구자/학자 | 논문 연결, 아이디어 발전 | Graph View, Thinking Commands |
| 개발자 | 기술 노트, 문제 해결 | Vim 모드, 코드블록, 검색 |
| 작가/소설가 | 세계관 구축, 캐릭터 연결 | WikiLinks, AI Synthesis |
| 기획자/PM | 아이디어 정리, 의사결정 | Ask My Brain, Knowledge Gap |

**Secondary (확장)**
- 학생, 창업가, 컨설턴트, 콘텐츠 크리에이터

### 경쟁 포지셔닝

```
옵시디언: 로컬 + 플러그인 생태계
노션: 팀 협업 + 데이터베이스
Second Brain: AI 지식 연결 + 웹 접근성

우리의 차별점:
1. AI 기능이 핵심 (노션 AI보다 깊은 지식 연결)
2. Graph View (노션에 없음)
3. 설치 없이 어디서든 (옵시디언 대비)
4. 개인 지식 관리 특화 (팀 협업 X)
```
