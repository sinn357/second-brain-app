## 구현 우선순위

### Phase 1: 즉시 처리 (핵심 UX)

> **목표**: 즉시성, 단순성, 신뢰성 기반 강화

| Task | 설명 | 카테고리 |
|------|------|----------|
| Home 페이지 개선 | 최신 메모 표시 + 바로 작성 가능 | 즉시성 |
| Cmd+N 전역 단축키 | 어느 화면에서든 새 메모 | 즉시성 |
| 저장 상태 표시 강화 | "저장됨 ✓" 명확하게, 실패 시 경고 | 신뢰성 |
| 테이블 기능 | Tiptap Table 확장 | 편집기 |
| 텍스트 하이라이트 | 배경색 강조 | 편집기 |

### Phase 2: 안정성 + 편의성

| Task | 설명 | 카테고리 |
|------|------|----------|
| 오프라인 모드 | ServiceWorker 캐싱, 복귀 시 동기화 | 신뢰성 |
| 갤러리 뷰 | 썸네일로 시각적 탐색 | 단순성 |
| 집중 모드 | 사이드바/메뉴 숨김, 에디터만 | 단순성 |
| 접기 섹션 | 헤딩 접기/펼치기 | 편집기 |
| 노트 잠금 | 개별 노트 비밀번호 | 보안 |
| 충돌 감지 | 탭 2개에서 동시 편집 시 경고 | 신뢰성 |

### Phase 3: 미디어 기본

| Task | 설명 | 카테고리 |
|------|------|----------|
| 이미지 첨부 | 드래그&드롭, 붙여넣기 | 미디어 |
| 파일 첨부 | PDF, 문서 등 | 미디어 |
| 링크 미리보기 | URL 입력 시 OG 이미지/제목 표시 | 미디어 |

### Phase 4: 확장 기능

| Task | 설명 | 카테고리 |
|------|------|----------|
| 녹음 기능 | 브라우저 MediaRecorder API | 오디오 |
| 음성 전사 | Whisper API 연동 | 오디오 |
| 비디오 첨부 | 비디오 업로드/임베드 | 미디어 |
| 위젯 | PWA 또는 네이티브 앱 필요 | 접근성 |

### Phase 5: 고급 기능 (장기)

| Task | 설명 | 카테고리 |
|------|------|----------|
| 스케치/손글씨 | Canvas 기반 드로잉 | 미디어 |
| 문서 스캔 | 카메라 + OCR | 미디어 |
| 수식 계산 | Math.js 또는 유사 라이브러리 | 편집기 |
| 공유 메모 | 링크로 공유 | 협업 |
| 실시간 협업 | WebSocket 기반 | 협업 |
