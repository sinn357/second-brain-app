# Second Brain App AI Features Ideas

> 세컨드브레인 앱의 지식 정리/연결/회상 기능을 강화하기 위한 AI 기능 아이데이션

**작성일**: 2026-02-02

---

## 1. 방향성

Second Brain App의 AI는 **노트 정리 자동화, 연결 강화, 회상/검색 최적화**에 집중한다.

핵심 목표:
- 지식 정리 비용 절감
- 노트 간 연결 강화
- 기억 회복(Recall) 효율 향상

---

## 2. 우선순위 기능 아이디어

### A. 노트 정리/요약
- **노트 요약 카드 자동 생성**
- **긴 노트 → 핵심 bullet 추출**
- **제목/태그/속성 자동 추천**

### B. 링크/연결 강화
- **관련 노트 자동 추천**
- **문맥 기반 [[note]] 자동 링크**
- **그래프에서 허브 노트 강조**

### C. 지식베이스화
- **노트 인사이트/정의/결론 추출 → 지식 카드화**
- **주제별 컬렉션 자동 생성**

### D. 검색/회상 지원
- **질문형 검색** (예: “지난주 책 요약 뭐였지?”)
- **기억 회복 패키지**: 관련 노트 + 요약 카드 제공

### E. 작업/프로젝트 연계
- **노트에서 TODO 추출 → 작업 생성**
- **회의 노트 → 액션 아이템 자동 추출**

### F. 작성 지원
- **Quick Add 확장**: 메모 → 정리된 노트
- **노트 템플릿 자동 적용** (회의록/리딩노트 등)
